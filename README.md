# HRMS Lite

A lightweight Human Resource Management System built using Django and React.

## Features

Employee Management
- Add Employee
- Delete Employee
- View Employee List

Attendance Management
- Mark Attendance
- View Attendance Records
- Filter Attendance by Date

Dashboard
- Total Employees
- Present Today
- Absent Today

## Tech Stack

Backend: Django
Frontend: React
Database: MySql
Deployment: Render + Vercel

## Local Setup

Backend

cd backend
pip install -r requirements.txt
python manage.py migrate
python manage.py runserver

Frontend

cd frontend
npm install
npm start

## Live Application

Frontend: Vercel URL  
Backend API: Render URL
